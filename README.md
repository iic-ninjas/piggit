# Piggit

A small git-like utility which grants you local version control for your PostgreSQL DB schema and data.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'piggit'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install piggit
